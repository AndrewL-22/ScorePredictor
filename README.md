## Features
- **Match Outcome Prediction:** Uses a custom algorithm to generate probabilities for match outcomes based on player performance metrics and historical match data.  
- **Live Data Integration:** Fetches and pre-processes live season data using API-Football to ensure accurate and up-to-date predictions.  
- **Android App Interface:** User-friendly interface built with Android Studio for seamless interaction on mobile devices.
